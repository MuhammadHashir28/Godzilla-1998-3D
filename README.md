# Godzilla 1998 3D Cinematic Landing Page

A fast, highly immersive, and cinematic React landing page inspired by the 1998 *Godzilla* film that uses 3d models to make the website modern. This project combines a dark, neon-green HUD aesthetic with responsive layouts, modular components, and cinematic media features.

## Features

* **Cinematic UI/UX:** High-contrast, dark-themed digital interface featuring sharp green HUD accents (`#22c55e` style matrix greens) and large movie-style cinematic visuals.
* **Modular Architecture:** Fully componentized layout including a dedicated audio widget, model stage configurations, and structured view sections.
* **Dedicated Data Views:** Includes content-focused sections such as **Awakening**, **Siege**, **Monster Profile** (with responsive stat blocks), **Gallery**, and an interactive **Timeline**.
* **Media Integration:** Integrated ambient music controls via a custom interactive audio widget and direct video trigger actions.

## Tech Stack

* **Frontend Library:** React.js , three.js
* **Language:** TypeScript / JavaScript (ES6+)
* **Styling:** Custom CSS3 (Modular design, responsive grid and flex layouts)
* **Build Tool:** Vite / Next-env configurations
